## Lambda 

```
npm install
```

Set the following env variables:

```
export aws_secret_access_key=
export aws_access_key_id=
export aws_account_id=
export aws_role=
export function_name=
export deployment_file=
export timeout=
export memory_size=
export runtime=
export publish=
export handler=
```
Then deploy to aws with:

```
gulp deploy
```
